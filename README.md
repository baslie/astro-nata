# astro-nata

Тестовый стенд для сравнения JS-библиотек натальных карт. Цель — оценить связку [CircularNatalHoroscopeJS](https://github.com/0xStarcat/CircularNatalHoroscopeJS) (астрологические расчёты) + [AstroChart](https://github.com/Kibo/AstroChart) (SVG-рендеринг) и определить дальнейшие шаги для продакшн-реализации.

## Что внутри

- **`index.html`** — форма ввода даты/времени/места рождения с автокомплитом городов (Geoapify)
- **`js/app.js`** — логика: расчёт натальной карты, маппинг планет/домов/углов, рендеринг SVG-чарта
- **`css/style.css`** — тёмная тема с CSS-переменными
- **`libs/CircularNatalHoroscopeJS/`** — библиотека расчётов (планеты, дома, аспекты, системы домов Placidus/Koch/Whole Sign и др.)
- **`libs/AstroChart/`** — библиотека SVG-рендеринга астрологических карт

## Функциональность

- Ввод даты, времени и места рождения
- Геокодинг с автоподстановкой городов и определением таймзоны
- Расчёт позиций планет (Солнце, Луна, Меркурий, Венера, Марс, Юпитер, Сатурн, Уран, Нептун, Плутон, Хирон)
- Лунные узлы и Лилит
- Асцендент и Мидхевен (MC)
- Отображение ретроградности
- Визуализация натальной карты в SVG с аспектами

## Технологии

- Vanilla JavaScript (без фреймворков)
- HTML5 / CSS3
- [CircularNatalHoroscopeJS](https://www.npmjs.com/package/circular-natal-horoscope-js) — астрономические и астрологические расчёты
- [AstroChart](https://github.com/Kibo/AstroChart) — SVG-рендеринг круговой карты
- [Geoapify Geocoder Autocomplete](https://www.geoapify.com/) — поиск городов и координат

## Запуск

Откройте `index.html` в браузере. Сборка не требуется — все зависимости подключены локально или через CDN.
